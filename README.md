# Game of Life

Implementation of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life).

Written in Rust using Ratatui for the Text User Interface (TUI).

## Run TUI

`cargo run --features="build-binary"`
